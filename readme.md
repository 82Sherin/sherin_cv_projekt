## ğŸ¨ CSS / Styling Notes

For this project, I have created separate CSS files for the **Home**, **Frontend/Backend**, and **Coordinator** pages.  
Each element was built individually this time. I am aware that there is some code repetition that could have been moved to a common root file, but as this is my first project, this approach helped me understand how each element works.

- On the **Home** page, I intentionally avoided using Flexbox to fully understand the basics of layout.  
- On the **Coordinator** page, most of the layout is built with **Flexbox**.  
- In the future, I plan to redo the **Hairstylist** and **Makeup Artist** sections by adding images directly instead of linking to external sites, and then build those layouts with **CSS Grid** for better structure and flexibility.

This project was a great learning experience to explore different layout techniques and understand the fundamentals of CSS.
